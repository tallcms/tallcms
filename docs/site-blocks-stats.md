# Stats Block

> Key metrics and statistics to highlight achievements and impact.

---

## Overview

The Stats block displays important numbers and metrics in an eye-catching format. Use it to showcase company achievements, impact metrics, or key figures.

---

## Fields

| Field | Description |
|-------|-------------|
| **Section Title** | Optional heading |
| **Stats** | Repeatable stat items |
| **Number** | The metric value |
| **Label** | Description of the metric |
| **Prefix** | Optional prefix (e.g., "$") |
| **Suffix** | Optional suffix (e.g., "+", "%") |
| **Animate** | Count-up animation on scroll |
| **Columns** | 2, 3, or 4 column layout |
