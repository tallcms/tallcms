# Features Block

> Grid layout showcasing product features, service highlights, or key benefits.

---

## Overview

The Features block displays a collection of features or benefits in a clean grid layout. Each feature can include an icon, title, and description.

---

## Fields

| Field | Description |
|-------|-------------|
| **Section Title** | Optional heading above the grid |
| **Section Description** | Optional intro text |
| **Features** | Repeatable items with icon, title, description |
| **Columns** | 2, 3, or 4 column layout |
| **Icon Style** | Colored, outlined, or filled |
| **Background** | Section background color |
