# Pricing Block

> Pricing tables with plan comparison for SaaS, services, or memberships.

---

## Overview

The Pricing block displays your pricing plans in a clear, comparable format. Highlight your recommended plan and optionally show monthly/yearly billing toggles.

---

## Fields

| Field | Description |
|-------|-------------|
| **Section Title** | Heading above pricing table |
| **Plans** | Repeatable plan items |
| **Plan Name** | Name of the tier |
| **Price** | Monthly/yearly price |
| **Features** | List of included features |
| **CTA Button** | Sign-up or purchase button |
| **Highlighted** | Mark as recommended plan |
| **Billing Toggle** | Show monthly/yearly switch |
