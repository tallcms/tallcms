# Logos Block

> Client or partner logo showcase for trust signals and social proof.

---

## Overview

The Logos block displays a collection of client, partner, or press logos. Use grayscale mode for a unified look that doesn't distract from your content.

---

## Fields

| Field | Description |
|-------|-------------|
| **Section Title** | Optional heading (e.g., "Trusted by") |
| **Logos** | Upload logo images |
| **Layout** | Grid or scrolling carousel |
| **Grayscale** | Display logos in grayscale |
| **Size** | Small, medium, or large logos |
| **Link** | Optional link for each logo |
