# Document List Block

> List of downloadable documents from media collections.

---

## Overview

The Document List block displays files from your media collections as downloadable links. Filter by file type and choose from list, card, or compact layouts.

---

## Fields

| Field | Description |
|-------|-------------|
| **Section Title** | Heading above document list |
| **Collections** | Select media collections |
| **File Types** | Filter: PDF, Word, Excel, ZIP, or all |
| **Layout** | List, cards, or compact |
| **Order** | Newest, oldest, or alphabetical |
| **Show File Size** | Display file size |
| **Show Type Badge** | Display file type icon/badge |
