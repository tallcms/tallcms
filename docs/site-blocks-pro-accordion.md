# Accordion Block

> Collapsible content sections perfect for FAQs, feature lists, and organizing information.

---

## Overview

The Accordion block creates expandable/collapsible sections that help organize large amounts of content in a compact, user-friendly format. Visitors can click items to reveal their content, keeping pages clean while maintaining access to detailed information.

---

## Fields

| Field | Description |
|-------|-------------|
| **Heading** | Optional section title above the accordion |
| **Subheading** | Supporting text below the heading |
| **Items** | Repeatable accordion entries with title and content |
| **Allow Multiple** | When enabled, multiple items can be open simultaneously |
| **First Open** | Automatically expand the first item on page load |
| **Style** | Visual style: Default, Bordered, or Minimal |
| **Content Width** | Control the maximum width of the content area |
| **Animation** | Entrance animation with optional stagger effect |

---

## Item Fields

Each accordion item includes:

| Field | Description |
|-------|-------------|
| **Title** | The clickable header text (required) |
| **Content** | The expandable body content (required) |

---

## Style Options

| Style | Description |
|-------|-------------|
| **Default** | Standard accordion with subtle separators |
| **Bordered** | Each item has a visible border |
| **Minimal** | Clean look without visual separators |
