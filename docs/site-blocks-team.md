# Team Block

> Team member profiles for about pages and company introductions.

---

## Overview

The Team block displays your team members with photos, names, roles, and optional bios and social links.

---

## Fields

| Field | Description |
|-------|-------------|
| **Section Title** | Heading above team grid |
| **Section Description** | Optional intro text |
| **Members** | Repeatable team member items |
| **Name** | Team member's name |
| **Role** | Job title or position |
| **Photo** | Profile picture |
| **Bio** | Short description |
| **Social Links** | LinkedIn, Twitter, etc. |
| **Columns** | 2, 3, or 4 column layout |
