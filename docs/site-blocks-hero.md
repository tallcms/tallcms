# Hero Block

> Full-width section perfect for page headers, landing pages, and promotional banners.

---

## Overview

The Hero block creates impactful, attention-grabbing sections at the top of your pages. It combines headlines, descriptions, call-to-action buttons, and background imagery to set the tone for your content.

---

## Fields

| Field | Description |
|-------|-------------|
| **Headline** | Main title text |
| **Description** | Supporting text below the headline |
| **Background Image** | Full-width background image |
| **Background Color** | Solid color alternative to image |
| **Overlay** | Darken or gradient overlay for text readability |
| **Primary Button** | Main CTA with text, link, and style |
| **Secondary Button** | Optional second CTA |
| **Text Alignment** | Left, center, or right |
| **Min Height** | Section height (small, medium, large, full-screen) |
