# Video Block

> Embed videos from YouTube, Vimeo, or self-hosted sources with full playback control.

---

## Overview

The Video block embeds videos from popular platforms or your own server. It supports YouTube, Vimeo, and self-hosted video files with customizable playback options, aspect ratios, and responsive sizing.

---

## Fields

| Field | Description |
|-------|-------------|
| **Heading** | Optional section title above the video |
| **Subheading** | Supporting text below the heading |
| **Source** | Video source: YouTube, Vimeo, or Self-hosted |
| **Aspect Ratio** | Video dimensions: 16:9, 4:3, 21:9, or 1:1 |
| **Width** | Maximum width: Full, XL, Large, or Medium |
| **Caption** | Optional caption text below the video |
| **Content Width** | Control the maximum width of the content area |
| **Animation** | Entrance animation effect |

---

## Source-Specific Fields

### YouTube

| Field | Description |
|-------|-------------|
| **YouTube URL** | Full YouTube video URL (required) |

### Vimeo

| Field | Description |
|-------|-------------|
| **Vimeo URL** | Full Vimeo video URL (required) |

### Self-Hosted

| Field | Description |
|-------|-------------|
| **Video URL** | Direct URL to MP4, WebM, or OGG file (required) |
| **Poster URL** | Thumbnail image shown before playback |

---

## Playback Options

| Option | Description |
|-------|-------------|
| **Autoplay** | Start playing automatically on page load |
| **Muted** | Start with audio muted (required for autoplay) |
| **Loop** | Continuously repeat the video |
| **Controls** | Show player controls (play, pause, volume, etc.) |

---

## Aspect Ratios

| Ratio | Best For |
|-------|----------|
| **16:9** | Standard widescreen (most common) |
| **4:3** | Traditional/older video formats |
| **21:9** | Cinematic ultra-wide content |
| **1:1** | Square format (social media style) |

---

## Tips

- For autoplay to work, the video must also be muted (browser requirement)
- YouTube and Vimeo handle their own responsive behavior
- Self-hosted videos should be optimized for web delivery
