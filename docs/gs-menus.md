# Set Up Navigation

> **What you'll learn:** How to create a navigation menu with links to your pages.

**Time:** ~5 minutes

---

## 1. Open the Menus panel

Navigate to **Admin > Content > Menus** in the sidebar.

## 2. Create a header menu

1. Click **New Menu**
2. Enter:
   - **Name**: "Main Navigation"
   - **Location**: Select "header"
3. Click **Create**

## 3. Add menu items

Click **Manage Items** on your new menu, then:

1. Click **New Item**
2. For a page link:
   - **Label**: "About"
   - **Type**: Page
   - **Page**: Select "About Us"
3. Click **Create**

Repeat to add more items (Home, Services, Contact, etc.).

## 4. Reorder items

Drag and drop menu items to reorder them. The order here matches your site navigation.

## 5. View your menu

Visit your site's frontend to see the menu in the header.

---

## Quick Tips

**Create nested menus**
Drag an item onto another to create a dropdown. Works great for "Services > Web Design, Services > Marketing".

**Add external links**
Set **Type** to "External" and enter the full URL (e.g., `https://twitter.com/yourhandle`).

**Create a footer menu**
Create another menu with **Location** set to "footer" for footer navigation.

---

## Common Pitfalls

**"Menu not appearing"**
Ensure the menu is set to **Active** and the **Location** matches where your theme displays menus.

**"Menu item not showing"**
Individual items can be toggled Active/Inactive. Check the item's status.

**"Page link shows wrong URL"**
Menu items pull the URL from the page slug. Edit the page to fix its slug.
