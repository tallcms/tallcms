# Timeline Block

> Chronological events or milestones for company history and roadmaps.

---

## Overview

The Timeline block presents events in chronological order. Use it for company history, project milestones, or product roadmaps.

---

## Fields

| Field | Description |
|-------|-------------|
| **Section Title** | Heading above timeline |
| **Events** | Repeatable timeline items |
| **Date** | Event date or period |
| **Title** | Event heading |
| **Description** | Event details |
| **Image** | Optional image |
| **Layout** | Vertical or alternating |
