# Testimonials Block

> Customer testimonials and reviews to build trust and social proof.

---

## Overview

The Testimonials block showcases customer feedback, reviews, and quotes. Display them as cards, a slider, or a grid layout.

---

## Fields

| Field | Description |
|-------|-------------|
| **Section Title** | Optional heading |
| **Testimonials** | Repeatable items |
| **Quote** | Customer's testimonial text |
| **Author Name** | Customer's name |
| **Author Photo** | Optional headshot |
| **Author Title** | Role or company |
| **Rating** | Optional star rating |
| **Layout** | Cards, slider, or grid |
