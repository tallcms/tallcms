# Parallax Block

> Full-width section with parallax scrolling effect.

---

## Overview

The Parallax block creates an immersive visual effect where the background image moves at a different speed than the foreground content as users scroll.

---

## Fields

| Field | Description |
|-------|-------------|
| **Background Image** | Image for parallax effect |
| **Headline** | Optional overlay text |
| **Description** | Optional supporting text |
| **Button** | Optional CTA button |
| **Overlay** | Darken image for text readability |
| **Height** | Section height |
| **Parallax Speed** | Intensity of the effect |
