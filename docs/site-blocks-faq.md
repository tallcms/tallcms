# FAQ Block

> Frequently asked questions in an accordion format.

---

## Overview

The FAQ block presents questions and answers in a collapsible accordion. Visitors can expand individual items to read the answers.

---

## Fields

| Field | Description |
|-------|-------------|
| **Section Title** | Heading above FAQ |
| **Section Description** | Optional intro text |
| **Questions** | Repeatable FAQ items |
| **Question** | The question text |
| **Answer** | Rich text answer |
| **Default Open** | Start with first item expanded |
| **Allow Multiple** | Allow multiple items open at once |
