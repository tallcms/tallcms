# Media Gallery Block

> Responsive image and video gallery with lightbox viewing.

---

## Overview

The Media Gallery block displays images and videos in a beautiful grid, masonry, or carousel layout. Click any item to view it in a full-screen lightbox with keyboard navigation.

---

## Fields

| Field | Description |
|-------|-------------|
| **Section Title** | Optional heading |
| **Media Source** | Manual upload or Media Collection(s) |
| **Media Type** | Images only, videos only, or both |
| **Layout** | Grid, masonry, or carousel |
| **Columns** | 2, 3, or 4 columns |
| **Lightbox** | Enable full-screen viewing |
| **Order** | Newest, oldest, or random (collection mode) |
| **Limit** | Maximum items to display |

---

## Manual vs Collection Mode

**Manual Upload:** Select specific images and videos for complete control over what appears.

**Collection Mode:** Select media collection(s) and the gallery automatically displays their contents. The gallery updates when you add media to the collection.
