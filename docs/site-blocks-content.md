# Content Block

> Rich text section for articles, descriptions, and general written content.

---

## Overview

The Content block is your go-to for adding text-based content to pages. It provides a rich text editor for formatting and supports various heading levels and content widths.

---

## Fields

| Field | Description |
|-------|-------------|
| **Title** | Section heading |
| **Heading Level** | H2, H3, or H4 |
| **Content** | Rich text body with formatting options |
| **Content Width** | Narrow, standard, wide, or full |
| **Background** | Optional background color |
