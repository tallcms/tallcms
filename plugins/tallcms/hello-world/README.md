# Hello World

A simple test plugin to demonstrate the plugin system

## Installation

### Via ZIP Upload
1. Download the plugin ZIP file
2. Go to Admin > Settings > Plugins
3. Click "Upload Plugin" and select the ZIP file

### Via File System
1. Extract the plugin to `plugins/tallcms/hello-world/`
2. The plugin will be auto-discovered
